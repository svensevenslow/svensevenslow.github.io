Study Materials

If you are just starting out as a computer programmer and have no prior programming experience
learning a language like python or C would be the best.Python will give you a feel of programming
and C will help clear your basics.

Game Development

If you are interested in learning about game development in Java or Android I would strongly recommend the tutorials written by James Cho

C programming

According to me C programming is the most important aspect to learn if you want to become a good programmer.The following book was a great help to me for learning C

    C How to Program by Paul J. Deitel and Harvey Deitel

Web Development

For learning the various aspects of Web-development like HTML,CSS and javascipt follow the Mozilla Developer Network .For additional help and references you could follow w3schools

Python

If you are complete beginner in computer programming and know nothing so far.It would be advisable to read the following

    Learn Python the hard way by Zed A. Shaw
If you have some base in programming and want to learn python reading the official documentation
would be the best.
